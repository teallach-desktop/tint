# Dependencies

- meson, ninja, cmake
- qt6-base, qt6-wayland
- [layer-shell-qt], [libsfdo]

[layer-shell-qt]: https://invent.kde.org/plasma/layer-shell-qt
[libsfdo]: https://gitlab.freedesktop.org/vyivel/libsfdo

# Build

```
meson setup build
meson compile -C build
```

# Run

```
build/tint -c doc/tintrc
```
